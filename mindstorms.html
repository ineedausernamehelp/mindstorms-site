Mindstorms Robot Behind the scenes

LEGO Mindstorms is een uitbreiding op normaal LEGO want je kan het prgrommeren. Het doel hiervan is programmeren leren aan nieuwe programmeurs. De LEGO robots bestaan uit sensors en moters die onder andere: Afstand , licht, rotatie, geluidsvolume, tempratuur en afstand kunnen waarnemen. Zodat de robot op invloeden van buiten kan waarnemen en daarop kan reageren. LEGO mindstorms kan op verschillende manier geprogrammeerd worden met python of woordblokken in dit project gebruiken wij python want woordblokken is niet een echte programeertaal.
